# LEONTIEVA — продюсерский центр

Официальный сайт продюсерского центра Елены Леонтьевой.

Основной адрес: [leontieva.media](https://leontieva.media)

## Направления

- медиадиагностика и позиционирование;
- стратегические сессии;
- подготовка к СМИ, ТВ и публичным выступлениям;
- продюсерское сопровождение бизнеса и экспертов.

## Разработка

```bash
npm install
npm run dev
npm test
```

Проект работает на vinext и публикуется через Sites. Контакт для новых обращений: `pr@leontieva-media.ru`.
