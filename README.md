# LEONTIEVA — продюсерский центр

Официальный сайт продюсерского центра Елены Леонтьевой.

Основной адрес: [leontieva.media](https://leontieva.media)

## Направления

- стратегические решения и дорожные карты;
- продюсерское и PR-сопровождение;
- медиа, видео и эфиры;
- общественные проекты;
- законный GR и работа со стейкхолдерами.

## Разработка

```bash
npm install
npm run dev
npm test
```

Основная публикация работает как статический сайт через GitHub Pages — по той же схеме, что и `leontev.pro`. Готовые `index.html`, `styles.css`, изображения и `CNAME` лежат в корне ветки `main`. Исходная версия на vinext сохранена для локальной разработки и предпросмотра.

Контакт для новых обращений: `pr@leontieva-media.ru`.
